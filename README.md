# hadoop-tech-stack
hadoop ecosystem technology stack conclusion
hadoop生态技术栈总结。
hadoop技术栈包含分布式存储、分布式计算，从hdfs，mapreduce衍生出众多技术产品。分布式存储：gfs，hdfs、alluxio等；分布式计算：mapreduce，spark等。基于分布式思想产生的分布式产品：bigtable，hbase，hive等。
